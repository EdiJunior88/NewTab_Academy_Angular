Instalação o Node.js

Antes de estudarmos mais sobre Angular, devemos ter instalado o Node.js em nosso sistema para configurarmos nosso ambiente de desenvolvimento.

Angular 8 requer Node.js. versão 10.9.0 ou posterior. Você pode baixá-lo em https://nodejs.org/en/

Após o download, você deve instalá-lo no seu sistema.

Veja como instalar o Node.js no Windows aqui: https://www.javatpoint.com/install-nodejs

Veja como instalar o Node.js no Linux / Ubuntu / CentOS aqui: https://www.javatpoint.com/install-nodejs-on-linux-ubuntu-centos

Depois de instalar o Node.js no seu sistema, abra o prompt / terminal.

Para verificar sua versão, execute o node -v e npm -v em uma janela de terminal / console.
