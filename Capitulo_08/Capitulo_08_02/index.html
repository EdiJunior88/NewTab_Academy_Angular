Instalação da CLI Angular

Use o seguinte comando para instalar a CLI Angular: 
    npm install -g @angular/cli
Ou: 
    npm install -g @angular/cli@latest

Ou ainda, basta ir ao site oficial da CLI Angular e seguir os passos: https://angular.io/cli

Você verá o comando CLI inteiro para criar um aplicativo Angular. Você precisa executar o primeiro comando para instalar o Angular CLI. Essas etapas são as mesmas para Windows e Mac.
