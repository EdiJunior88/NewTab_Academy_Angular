Primeiro Aplicativo Angular 8

Vamos ver como criar um aplicativo Angular 8. Para isso, utilize:

Sintaxe:

    ng new app_name

Por exemplo: aqui, vamos criar um aplicativo chamado "angular8firstapp"

Você pode ver que uma pasta foi criada. Este é o seu primeiro aplicativo criado do Angular 8.
