Arquivos usados ​​na pasta

Os arquivos do Angular usados ​​principalmente em seu projeto são estes abaixo:

pasta src: esta é a pasta que contém os principais arquivos de código relacionados ao seu aplicativo angular.

pasta app: contém os arquivos que você criou para os componentes do aplicativo.

app.component.css: este arquivo contém o código das folhas de estilo em cascata para o componente do aplicativo.

app.component.html: este arquivo contém o arquivo html relacionado ao componente do aplicativo. Este é o arquivo de modelo usado pelo angular para fazer a ligação de dados.

app.component.spec.ts: este arquivo é um arquivo de teste de unidade relacionado ao componente do aplicativo. Este arquivo é usado junto com outros testes de unidade. É executado a partir da CLI Angular pelo comando ng test.

app.component.ts: este é o arquivo de texto mais importante, que inclui a lógica de exibição por trás do componente.

app.module.ts: também é um arquivo de texto que inclui todas as dependências do site. Este arquivo é usado para definir os módulos necessários a serem importados, os componentes a serem declarados e o componente principal a ser inicializado.
