Módulos

O módulo refere-se a um local onde você pode agrupar os componentes, diretivas, pipes e serviços relacionados ao aplicativo. Caso você esteja desenvolvendo um site, o cabeçalho, rodapé, se tornam parte de um módulo.

Para definir o módulo, podemos usar o NgModule. Quando você cria um novo projeto usando o comando Angular –cli, o NgModule é criado no arquivo app.module.ts por padrão e tem a seguinte aparência:

O NgModule precisa ser importado da seguinte maneira:

    import { NgModule } from '@angular/core';

A estrutura para o NgModule é mostrada abaixo:

Começa com @NgModule e contém um objeto que possui declarations, imports, providers e bootstrap.

    Declarations

É uma matriz de componentes criados. Se qualquer novo componente for criado, ele será importado primeiro e a referência será incluída nas declarações, como mostrado abaixo:

    declarations: [
      AppComponent,
      NewCmpComponent
    ]

    Imports

É uma matriz de módulos que devem ser utilizados no aplicativo. Por exemplo, agora no @NgModule, vemos o Módulo do navegador importado (BrowserModule).

    Providers

Isso incluirá os serviços criados.

    Bootstrap

Isso inclui o principal componente do aplicativo para iniciar a execução.
