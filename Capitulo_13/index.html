Diretivas

As diretivas são usadas para manipular o DOM. Usando diretivas, você pode alterar a aparência, o comportamento ou o layout de um elemento. Também ajuda a estender o HTML.

As diretivas podem ser classificadas em 3 categorias, com base em como elas se comportam:

  • Diretivas de componentes.
  • Diretivas estruturais.
  • Diretivas de atributos.

Diretivas de componentes: são usadas na classe principal. Eles contêm os detalhes de como o componente deve ser processado, instanciado e usado em tempo de execução.

Diretivas estruturais: começam com um sinal *. Essas diretivas são usadas para manipular e alterar a estrutura dos elementos DOM. Por exemplo, diretiva *ngIf, *ngSwitch e *ngFor.

  • *ngIf: O ngIf nos permite adicionar / remover elemento DOM.
  • *ngSwitch: Nos permite adicionar / remover elementos.
  • *ngFor: é usada para repetir uma parte do modelo HTML uma vez por cada item de uma lista iterável (Coleção).

Diretivas de atributo: são usadas para alterar a aparência e o comportamento dos elementos. Por exemplo: ngClass e ngStyle etc.

  • ngClass: é usada para adicionar ou remover classes CSS a um elemento HTML.
  • ngStyle: facilita a modificação do estilo de um elemento HTML usando a expressão Você também pode usar a diretiva ngStyle para alterar dinamicamente o estilo do seu elemento HTML.
