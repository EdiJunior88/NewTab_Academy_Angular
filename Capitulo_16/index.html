Como usar *ngIf *ngFor *ngSwitch

NgIf

São usadas para adicionar ou remover elementos HTML de acordo com a expressão. A expressão deve retornar um valor booleano. Se a expressão for falsa, o elemento será removido, caso contrário, o elemento será inserido.

    <p *ngIf="condition">
      Olá mundo!
    </p>

NgFor

A diretiva *ngFor é usada para repetir uma parte do modelo HTML uma vez por cada item de uma lista iterável (Coleção). Algumas variáveis locais como Index, First, Last, odd e even são exportadas pelo *ngFor.

Veja a sintaxe simplificada para a diretiva ngFor:

    <li *ngFor="let item of items;"> …. </li>

NgSwitch

NgSwitch é uma diretiva estrutural usada para adicionar / remover elemento DOM. A diretiva ngSwitch é aplicada ao elemento container com uma expressão switch.

    <container_element [ngSwitch]="switch_expression">
      <inner_element *ngSwitchCase="match_expresson_1">…</inner_element>
      <inner_element *ngSwitchCase="match_expresson_2">…</inner_element>
      <inner_element *ngSwitchCase="match_expresson_3">…</inner_element>
      <inner_element *ngSwitchDefault>…</element>
    </container_element>

NgSwitchDefault

Você também pode aplicar a diretiva ngSwitchDefault no Angular 8. O elemento com ngSwitchDefault será exibido apenas se nenhuma correspondência for encontrada. O elemento interno com ngSwitchDefault pode ser colocado em qualquer lugar dentro do elemento container e não necessariamente na parte inferior. Se você adicionar mais de uma diretiva ngSwitchDefault, todos eles serão exibidos.
