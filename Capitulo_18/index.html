Ligação de dados (databinding)

A ligação de dados é o conceito principal do Angular e é usado para definir a comunicação entre um componente e o DOM. É uma técnica para vincular seus dados à sua camada de visualização. Em palavras simples, você pode dizer que a ligação de dados é uma comunicação entre o código de texto do seu componente e o modelo que o usuário vê. Facilita a definição de aplicativos interativos sem se preocupar em enviar e receber dados.

A ligação de dados pode ser unidirecional ou bidirecional.

Ligação de dados unidirecional

A ligação de dados unidirecional é uma comunicação unidirecional simples, na qual o modelo HTML é alterado quando fazemos alterações no código TypeScript. Angular fornece três tipos de ligação de dados unidirecional:

  • Interpolação de String
  • Vinculação de Propriedade
  • Ligação de evento

Interpolação de string

A interpolação de sequência de caracteres é uma técnica de ligação de dados unidirecional que é usada para gerar dados de um código TypeScript para um modelo HTML (exibição). Ele usa a expressão do modelo entre chaves duplas para exibir os dados do componente para a visualização.

Por exemplo:

    {{data}}

A interpolação de string adiciona o valor de uma propriedade do componente.

Sintaxe:

    <li>Name: {{ user.name }}</li>
    <li>Email: {{ user.email }}</li>

Vinculação de Propriedade

A vinculação de propriedades também é uma técnica de ligação de dados unidirecional. Na ligação de propriedades, vinculamos uma propriedade de um elemento DOM a um campo que é uma propriedade definida em nosso código TypeScript do componente.

Por exemplo:

    <img [src] = "imgUrl" />

Sintaxe:

    <input type="email" [value]="user.email">

Ligação de evento

No Angular é usada para manipular os eventos gerados no DOM, como clique no botão, movimento do mouse etc. Quando o evento DOM acontece (por exemplo, clique, alterar, keyup), ele chama o método especificado no componente. No exemplo a seguir, o método calcular() do componente é chamado quando o botão é clicado:

Por exemplo:

    <button (click)="calcular()"></button>

Ligação de dados bidirecional

Na ligação de dados bidirecional, a sincronização automática de dados ocorre entre o Modelo e a Visualização. Aqui, a mudança é refletida nos dois componentes. Sempre que você fizer alterações no modelo, ele será refletido na visualização e quando você fizer alterações no modo de exibição, será refletido no modelo.

Isso acontece imediata e automaticamente, garante que o modelo HTML e o código TypeScript sejam atualizados o tempo todo.

Por exemplo:

    [(ngModel)] = "[property of your component]"
